🚀 Project Overview

This project presents an interactive Tableau Dashboard built using the Superstore dataset. The objective is to analyze sales performance, profitability, customer behavior, product performance, shipping efficiency, and return trends to support data-driven business decisions.

The dashboard provides a comprehensive view of business operations through KPIs, trend analysis, geographical insights, customer segmentation, and operational metrics.

🎯 Business Objectives

✅ Monitor overall business performance

✅ Analyze sales and profit trends

✅ Identify top-performing products and categories

✅ Understand customer purchasing behavior

✅ Evaluate regional and state-wise performance

✅ Analyze the impact of discounts on profitability

✅ Monitor product returns and shipping efficiency
